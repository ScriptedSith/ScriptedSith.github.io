Event Protocols

Infestation

• Announce the event start in-game.
• Define the kill requirement for promotion (2-7 kills).
• Execute the command: !Infest

All command usage is logged. Unauthorized use or spamming will result in immediate demotion.

Mosh Pit / Randomizer

• Announce the event and duration.
• Set promotion requirements (1-3 kills).
• Clarify if spectators are permitted.
• Commands: !MoshPit or !MoshPitRandom

All command usage is logged. Unauthorized use or spamming will result in immediate demotion.

Terminal Raid

• Announce the raid in-game.
• All attendees pass upon completion.
• Run the cycle 3-5 times.
• Execute the command: !raid

Raid Boss

• Announce the boss encounter.
• All attendees pass upon completion.
• Run the cycle 3-5 times.
• Setup: Manually have the "Boss" player switch to the Hostile team.

VIP Transport

• Announce start/end points of the route.
• Split attendees into two groups (Escort vs. Assassins).
• All attendees pass; run 3-5 times.

Hide and Hunt / Manhunt

• Hide & Hunt: 1-3 mins to hide, 4-5 mins to hunt. Randomly pick a hunter.
• Manhunt: Set a specific end-goal/extraction point for the hiders to reach.
• All attendees pass; run 3-5 times.
