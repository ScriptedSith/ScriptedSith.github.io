Trials of Ascension

Apprentice Trial

Requirement: Must have been an apprentice for a minimum of 3 days. • Recite the Sith Code line by line via whisper.
• The Master chooses the specific format of the trial (Combat, Lore, or Task).
• Upon passing, the apprentice is promoted to Sith Knight and may choose their future path.

Knight Trial

Requirement: Sith Eradicator rank. Hosted by Sith Master+. • Recite the Sith Code with zero mistakes.
• Answer 2 Lore Questions (1 mistake allowed).
• Answer 3 IP Questions (1 mistake allowed).
• Duel the host 3 times.

POST-TRIAL: Announce KOS for 3 minutes.

Overseer Trial

Requirement: Sith Knight rank. Hosted by Sith Lord+. • Recite the Sith Code with zero mistakes.
• Answer 5 Lore Questions (2 mistakes allowed).
• Answer 5 IP Questions (2 mistakes allowed).
• Duel the host 7 times.

POST-TRIAL: Announce KOS for 5 minutes.

Master Trial

Requirement: Sith Overseer rank. Hosted by Sith Lord+. • Recite the Sith Code with zero mistakes.
• Answer 7 Lore Questions (2 mistakes allowed).
• Answer 7 IP Questions (2 mistakes allowed).
• Duel the host 10 times.

POST-TRIAL: Announce KOS for 8 minutes.

Lord Trial

Requirement: Sith Master rank. Hosted by Darth+. • Recite the Sith Code with zero mistakes.
• Answer 9 Lore Questions (1 mistake allowed).
• Answer 9 IP Questions (1 mistake allowed).
• Duel the host 15 times.

POST-TRIAL: Announce KOS for 10 minutes.

Darth Trial

Requirement: Request vote; must gain 2/3 majority from Dark Council+. • Recite the Sith Code with zero mistakes.
• Answer 10 Lore Questions (Zero mistakes allowed).
• Answer 10 IP Questions (Zero mistakes allowed).
• Duel the host 20 times.

POST-TRIAL: Announce KOS for 15 minutes.
