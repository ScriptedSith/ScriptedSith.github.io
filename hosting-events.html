Events

Infestation

Start off by announcing in-game.
State the number of kills needed to be promoted. (Between 2-7)
Begin the event with !Infest

All uses of the command are logged and anytime the command is sent. If the command is used without the event being hosted or is used without permission on spam you will be demoted.

All attendees pass.

Give 4-5 minutes to hunt.
Run 3-5 times.
